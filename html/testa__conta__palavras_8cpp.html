Detailed Description

Arquivo de testes para as funções de processamento de palavras.

Este arquivo contém os testes das funções do programa conta_palavras.cpp. Ele utiliza o framework Catch2 para testar as funcionalidades como abertura de arquivos, leitura de conteúdo, separação de palavras, contagem de palavras, e ordenação das palavras.
